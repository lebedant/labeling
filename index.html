Labeling user study

You will be presented with a labeled image. Please proceed as follows:

  • Examine the image and for each label try to determine to which part of the depicted object it belongs.
  • Then press "Start test" button and wait. One of the object parts will be highlighted in light green.
  • Click on the label that belong to the highlighted object part as quickly as possible, but at the same time try to be sure that the label belongs to the highlighted object part. If you cannot decide which label belongs to the highlighted object part press "I cannot decide" button. If you think that there is no label for the highlighted object part press "There is no label" button.
  • Wait until another object part is highlighted. And continue as before.
  • The whole test will be repeated on four different labeled images.
Thank you for your cooperation!
